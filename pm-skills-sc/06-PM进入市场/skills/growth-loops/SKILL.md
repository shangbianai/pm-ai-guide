# 增长环 (Growth Loops)

识别增长环（飞轮）以实现可持续增长。评估 5 种循环类型：病毒式、使用型、协作型、UGC 和推荐型。

## 5 种增长环类型

### 1. 病毒环 (Viral Loop)
用户作为使用产品的自然过程带入新用户（例如，邀请同事加入 Slack 工作区）。

### 2. 使用环 (Usage Loop)
用户使用产品越多，它对其他人的价值就越大（例如，共享数据库或网络效应）。

### 3. 协作环 (Collaboration Loop)
当用户一起工作时，产品的价值会增加（例如，Google Docs 中的实时编辑）。

### 4. UGC 环 (用户生成内容)
用户创建内容，通过 SEO 或社交分享吸引新用户（例如，Pinterest、Quora 或 Stack Overflow）。

### 5. 推荐环 (Referral Loop)
激励用户邀请他人（例如，Dropbox 的“获取更多空间”计划）。

## 衡量循环有效性

-   **循环系数 (k):** 每个现有用户平均带入的新用户数。
-   **周期时间 (Cycle Time):** 用户完成一次循环所需的时间。
-   **转化率 (Conversion Rate):** 被邀请/接触的用户中注册的百分比。

## 工作流程

1.  **审计现有的增长渠道**。
2.  **识别符合你产品核心价值的潜在循环**。
3.  **绘制出每个潜在循环的步骤**。
4.  **优先选择 1-2 个循环**进行实验。
5.  **定义指标**来跟踪循环健康状况。
6.  **通过减少每个步骤的摩擦来优化循环**。
