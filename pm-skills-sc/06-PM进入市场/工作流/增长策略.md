# /growth-strategy -- 增长环与 GTM 方式 (Growth Loops & GTM Motions)

识别并设计驱动可持续增长的机制。评估五种增长环类型和七种 GTM 方式，以构建平衡的获取和扩张策略。

## 调用示例

```
/growth-strategy B2B 协作工具 —— 用户增长停滞在 5K
/growth-strategy 寻求病毒式增长的消费级健身应用
/growth-strategy [上传产品指标或增长数据]
```

## 工作流程

### 第 1 步：理解增长背景

询问：
- 产品是什么？谁在使用它？
- 当前增长指标：用户数、增长率、获取渠道
- 什么有效？什么无效？
- 商业模式：收入与用户增长的关系是什么？
- 团队和预算：你可以为增长投入哪些资源？

### 第 2 步：评估增长环 (Growth Loops)

应用 **growth-loops** 技能：

分析适合你产品的五种增长环类型：

1. **病毒环 (Viral Loop)**：用户作为产品自然使用的一部分邀请他人
2. **使用环 (Usage Loop)**：更多使用创造更多价值，吸引用户回归
3. **协作环 (Collaboration Loop)**：与他人一起使用时，产品变得更有价值
4. **用户生成内容环 (UGC Loop)**：用户创建的内容吸引新用户
5. **推荐环 (Referral Loop)**：满意用户主动推荐给他人

针对每个适用的循环：机制、要求、预期影响、实施难度。

### 第 3 步：评估 GTM 方式 (GTM Motions)

应用 **gtm-motions** 技能：

评估七种 GTM 方法：

1. **入站 (Inbound)**：内容、SEO、思想领导力
2. **出站 (Outbound)**：销售、冷触达、基于账户的销售
3. **付费数字广告 (Paid Digital)**：SEM、社交广告、展示广告、重定向
4. **社区 (Community)**：论坛、活动、用户组、开发者关系
5. **合作伙伴 (Partners)**：集成商、转售商、联合营销
6. **ABM (基于账户的营销)**：针对性的企业级获取
7. **PLG (产品驱动增长)**：免费层级、自服务、产品病毒性

针对每种方式：与产品的契合度、预期 CAC、见效时间、所需工具。
