# /stakeholder-map -- 利益相关者地图与沟通计划

识别项目的所有利益相关者，按影响力和利益进行分类，并生成沟通计划，确保正确的人在正确的时间获得正确的信息。

## 调用示例

```
/stakeholder-map 新分析平台发布
/stakeholder-map 影响所有客户的定价模型变更
/stakeholder-map [上传项目简报或组织架构图]
```

## 工作流程

### 第 1 步：了解倡议背景

询问：
- 项目或倡议是什么？
- 当前处于哪个阶段？（计划、构建、发布、发布后）
- 哪些是你已经知道的显而易见的利益相关者？
- 是否有任何需要注意的敏感政治动态？

### 第 2 步：识别利益相关者

头脑风暴用户可能未考虑到的利益相关者：
- **内部**: 工程、设计、QA、数据、法务、财务、市场、销售、支持、领导层
- **外部**: 客户、合作伙伴、供应商、监管机构、董事会成员
- **经常被忽视**: 相邻团队、值班工程师、客户成功团队、文档团队

### 第 3 步：绘制权力/利益矩阵

应用 **stakeholder-map** 技能：

将每个利益相关者放入对应象限：

```
                    高利益 (HIGH INTEREST)
                         │
    令其满意             │      重点管理
    (高权力，            │      (高权力，
     低利益)             │       高利益)
                         │
   ──────────────────────┼──────────────────────
                         │
    监督                 │      随时告知
    (低权力，            │      (低权力，
     低利益)             │       高利益)
                         │
                    低利益 (LOW INTEREST)
```

### 第 4 步：生成沟通计划

```
## 利益相关者地图: [倡议项目名称]

### 利益相关者矩阵
| 利益相关者 | 角色 | 权力 | 利益 | 象限 | 立场 |
|------------|------|-------|----------|----------|--------|

### 沟通计划

#### 重点管理 (Manage Closely - 高权力，高利益)
| 利益相关者 | 渠道 | 频率 | 内容 | 负责人 |
|------------|---------|-----------|---------|-------|

#### 令其满意 (Keep Satisfied - 高权力，低利益)
| 利益相关者 | 渠道 | 频率 | 内容 | 负责人 |
|------------|---------|-----------|---------|-------|

#### 随时告知 (Keep Informed - 低权力，高利益)
| 利益相关者 | 渠道 | 频率 | 内容 | 负责人 |
|------------|---------|-----------|---------|-------|

#### 监督 (Monitor - 低权力，低利益)
[最低限度的沟通 —— 仅包含在大范围更新中]

### 潜在冲突
[利益相关者利益可能发生冲突的地方 —— 以及缓解策略]

### 升级路径
[当决策受阻时应该向谁寻求帮助]

### RACI 矩阵
| 决策领域 | 负责 (R) | 问责 (A) | 咨询 (C) | 知会 (I) |
|--------------|-------------|-------------|-----------|----------|
```

保存为 markdown 文件。

### 第 5 步：后续建议

- "需要我为 '重点管理' 组 **起草第一份利益相关者更新** 吗？"
- "我是否应该为关键利益相关者的对话 **创建会议准备简报**？"
- "需要我将沟通频率 **设置为定期检查清单** 吗？"

## 注意事项

- "重点管理" 象限是 PM 投入政治资本最多的地方 —— 务必处理好这些关系
- "立场"（支持、中立、抵触）有助于确定关系维护的优先级
- 不要忘记下游利益相关者：支持、文档和销售赋能团队往往会对突如其来的发布感到意外
- 随着项目演进更新地图 —— 利益相关者的利益会随项目阶段而变化
