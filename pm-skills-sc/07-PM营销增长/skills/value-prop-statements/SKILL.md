# 价值主张陈述
从现有的价值主张中，为市场营销、销售和新手引导生成价值主张陈述。

在编写营销文案、创建销售消息或制作新手引导消息时使用。

## 任务说明
- 使用此 [示例](https://www.canva.com/pro/) 作为参考。
- 至少生成 **3 个** 价值主张陈述。
- 确保每个陈述都清晰、简洁，并侧重于用户利益。
- 针对不同的受众：市场营销（潜在客户）、销售（潜在买家）和新手引导（新用户）。
