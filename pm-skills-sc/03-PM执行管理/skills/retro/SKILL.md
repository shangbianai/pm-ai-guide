# 冲刺回顾
对 $ARGUMENTS 的冲刺进行反思。

### 回顾问题
1. **哪些地方做得好？**: 哪些我们应该继续保持？
2. **哪些地方可以改进？**: 哪些没有按计划进行？
3. **我们学到了什么？**: 有什么新的见解或发现？
4. **待办事项**: 下个冲刺我们将做出哪些改变？

### 指导原则
- 创造一个安全的环境，鼓励公开和诚实的反馈。
- 专注于流程和系统，而不是个人。
- 确保待办事项是具体的、可衡量的，并分配给团队成员。
