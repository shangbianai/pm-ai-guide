# 05-PM数据分析 (PM Data Analytics)

产品经理数据分析技能库：涵盖 SQL 查询生成、同期群分析和 A/B 测试分析。

## 技能 (Skills) (3)

- **ab-test-analysis** — 分析具有统计显著性的 A/B 测试结果，包括样本量验证、置信区间和可操作的建议。
- **cohort-analysis** — 对用户参与度数据进行留存分析。
- **sql-queries** — 根据自然语言描述生成 SQL 查询。

## 工作流 (Workflows) (3)

- `A-B测试分析.md` — 分析 A/B 测试结果 —— 统计显著性、样本量验证以及发布/延长/停止建议。
- `同期群分析.md` — 对用户数据进行留存分析 —— 留存曲线、功能采用和参与趋势。
- `编写SQL查询.md` — 根据自然语言生成 SQL 查询 —— 支持 BigQuery、PostgreSQL、MySQL 等。
