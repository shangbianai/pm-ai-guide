# 利益相关者地图 (Stakeholder Mapping)

识别并分类 $ARGUMENTS 的利益相关者。

### 权力 × 利益矩阵 (Power × Interest Matrix)
| | 高利益 | 低利益 |
|---|---|---|
| **高权力** | **重点管理 (Manage Closely)**: 核心人物，定期沟通。 | **令其满意 (Keep Satisfied)**: 高影响力，满足其需求。 |
| **低权力** | **随时告知 (Keep Informed)**: 支持者/感兴趣者，保持更新。 | **监督 (Monitor)**: 投入最少精力，监控变化。 |

### 沟通计划
- **利益相关者**: 是谁？
- **象限**: 他们在矩阵中的哪个位置？
- **频率**: 我们多久沟通一次？
- **渠道**: 我们如何沟通（例如：Slack、电子邮件、会议）？
