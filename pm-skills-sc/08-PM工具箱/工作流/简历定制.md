# /tailor-resume -- 简历与职位描述 (JD) 优化

根据您的简历和目标职位描述，战略性地对齐您的经验，以最大限度地增加面试机会。关键词优化、要点重写和差异分析。

## 调用方式

```
/tailor-resume [上传简历] 这是 JD：[粘贴职位描述]
/tailor-resume [以文件形式上传简历和 JD]
```

## 工作流程

### 第 1 步：接收两个文档

需要两个输入：
- 简历（文本、PDF 或 DOCX）
- 目标职位描述（文本、URL 或文件）

如果只提供了一个，请索要另一个。

### 第 2 步：分析职位描述

提取：
- 必备任职资格和技能
- 优先任职资格
- 核心职责
- 行业和领域信号
- 资历级别指标
- 文化和团队信号

### 第 3 步：量身定制简历

应用 **review-resume** 技能：

- **关键词对齐**：将 JD 关键词映射到简历内容，自然地添加缺失的关键词
- **要点重写**：使用 XYZ+S 公式重塑经验，以强调与 JD 相关的成就
- **章节重排**：优先展示最相关的经验
- **个人总结/目标**：重写以直接针对该角色
- **技能章节**：与 JD 要求保持一致

### 第 4 步：生成定制简历 + 分析报告

```
## 简历定制：[公司名称] 的 [职位名称]

### 匹配度得分：[X/10]

### 关键词差异分析
| JD 关键词 | 简历中已有？ | 建议 |
|-----------|-----------|---------------|

### 已做出的更改
1. **[章节]**：[更改了什么以及原因]
2. ...

### 定制简历文本
[完整重写的简历文本]

### 差异分析
**强匹配项**：[您的经验直接对应的部分]
**重塑匹配项**：[为了适应 JD 而重新定位的经验]
**差距**：[您没有明确涉及的 JD 要求 —— 以及建议]

### 求职信要点
[在求职信中需要强调的 3-4 个要点，用以弥补剩余的差距]
```

将定制后的简历保存为 markdown。

## 注意事项

- 严禁捏造经验 —— 在真实的基础上重塑，不要凭空发明
- 个人总结/目标是每次申请中投资回报率 (ROI) 最高的可定制部分
- 尽可能精确地匹配 JD 的语言（如果他们说“跨职能 (cross-functional)”，就使用“跨职能”）
- 对于高级职位，强调规模和战略影响力；对于个人贡献者 (IC) 职位，强调动手执行能力
