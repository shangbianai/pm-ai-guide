# SWOT 分析 (SWOT Analysis)

## 元数据
- **名称**: swot-analysis
- **描述**: 进行全面的 SWOT 分析，识别内部因素（优势、劣势）和外部因素（机会、威胁）。

## 说明
当用户要求进行 SWOT 分析时，请提供以下结构的分析：

1. **优势 (Strengths) - 内部因素**:
   - 我们的产品做得特别好的地方是什么？
   - 我们拥有哪些独特资源？
   - 客户认为我们的主要优势是什么？

2. **劣势 (Weaknesses) - 内部因素**:
   - 我们在哪些方面可以改进？
   - 我们缺乏哪些资源或专业知识？
   - 客户最常见的抱怨是什么？

3. **机会 (Opportunities) - 外部因素**:
   - 我们面前有哪些有利趋势？
   - 我们可以利用哪些市场差距？
   - 我们可以采用哪些新技术？

4. **威胁 (Threats) - 外部因素**:
   - 竞争对手在做什么？
   - 市场法规或经济环境有哪些变化？
   - 哪些劣势可能使我们面临风险？

5. **战略建议 (Strategic Recommendations)**:
   - **优势-机会 (S-O)**: 如何利用优势来抓住机会？
   - **劣势-机会 (W-O)**: 如何通过利用机会来克服劣势？
   - **优势-威胁 (S-T)**: 如何利用优势来减少威胁的影响？
   - **劣势-威胁 (W-T)**: 如何尽量减少劣势并避开威胁？

## 输出格式
使用 2x2 的 Markdown 表格展示 SWOT 矩阵，随后提供详细的建议。

## 示例提示
- "对 [产品名称] 进行 SWOT 分析。"
- "分析我们进入 [市场名称] 的机会和威胁。"
- "为我们的新 AI 功能进行 SWOT 评估。"
