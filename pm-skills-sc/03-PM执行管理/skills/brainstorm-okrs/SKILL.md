# OKR 头脑风暴

为 $PRODUCT 生成具有挑战性的目标 (Objectives) 和可衡量的关键结果 (Key Results)。

### 目标公式
**动词 + 你想要实现的目标 + 业务价值**
*示例："通过提供最直观的结账体验来主导 SMB 市场。" *

### 关键结果公式
**动词 + 指标 + 从 X 提高到 Y**
*示例："将结账转化率从 15% 提高到 25%。" *

### 指导原则
- **目标 (Objectives)** 应该是定性的、鼓舞人心的且有时间限制的。
- **关键结果 (Key Results)** 必须是定量的、可衡量的且以结果为导向的。
- 专注于 **产出结果 (Outcomes)** (影响力) 而非 **产出物 (Outputs)** (任务)。
