# /market-scan -- 宏观环境分析

运行多个战略分析框架，以了解您的竞争和宏观环境。将 SWOT、PESTLE、波特五力模型和安索夫矩阵结合到一个单一的战略概览中。

## 调用方式

```
/market-scan 企业培训的教育科技 (EdTech) 市场
/market-scan [上传市场简报或战略文档]
/market-scan 我们的金融科技产品 —— 为董事会战略审查做准备
```

## 工作流程

### 第 1 步：理解上下文

询问：
- 您正在分析什么产品、公司或市场？
- 目的是什么？（战略规划、市场进入、投资者准备、年度审查）
- 您想重点关注哪些特定框架？还是运行全部四个？
- 您目前在这个市场中的地位如何？

### 第 2 步：运行分析

按顺序应用四个技能，每个技能都建立在之前技能的见解之上：

**SWOT 分析** (应用 **swot-analysis** 技能)：
- 内部：优势 (Strengths) 和劣势 (Weaknesses)
- 外部：机会 (Opportunities) 和威胁 (Threats)
- 为每个象限提供可操作的建议

**PESTLE 分析** (应用 **pestle-analysis** 技能)：
- 政治 (Political)、经济 (Economic)、社会 (Social)、技术 (Technological)、法律 (Legal)、环境 (Environmental) 因素
- 对每个因素进行影响评估和时间线预测

**波特五力模型** (应用 **porters-five-forces** 技能)：
- 现有竞争者的竞争、供应商的议价能力、购买者的议价能力、替代品的威胁、潜在进入者的威胁
- 整体行业吸引力评级

**安索夫矩阵** (应用 **ansoff-matrix** 技能)：
- 市场渗透、市场开发、产品开发、多元化
- 风险调整后的增长机会

### 第 3 步：综合分析

跨框架交叉引用发现结果，以识别：
- **趋同信号**：多个框架达成一致的方面
- **战略要务**：在各项分析中都显得至关重要的行动
- **关键风险**：需要缓解的威胁和力量
- **增长机会**：最佳风险调整后的机会所在

### 第 5 步：提供后续步骤

- “想让我根据这些发现**构建产品策略**吗？”
- “我应该**分析波特分析中识别出的特定竞争对手**吗？”
- “想让我为市场渗透机会**设计定价策略**吗？”

## 注意事项

- 使用网络搜索将分析建立在当前市场数据的基础上，而不仅仅是通用知识
- PESTLE 因素应包括具体的法规、市场数据和趋势信号 —— 而非通用的观察
- 波特五力模型在识别出*具体*的作用力时最有用，而不仅仅是抽象地对它们进行评级
- 安索夫矩阵应包含具体的增长点，而不仅仅是通用的“进入新市场”
- 综合分析部分是最有价值的部分 —— 它是各个框架相互对话的地方
