# 理想客户画像 (ICP)

从研究数据中识别理想客户画像 (ICP)，包含人口统计、行为、JTBD 和需求。在定义 ICP、分析 PMF 调查数据或了解谁是你最好的客户时使用。

## ICP 的关键组成部分

### 1. 人口统计 / 公司特征 (Demographics / Firmographics)
-   **行业：** 他们在哪些行业运营？
-   **规模：** 公司规模（收入、人数）。
-   **地理位置：** 他们位于哪里？
-   **角色：** 关键决策者和影响力者。

### 2. 行为特征
-   他们已经在使用什么工具？
-   他们在网上哪里出没？
-   他们如何消费信息？

### 3. 工作任务 (Jobs-to-be-Done - JTBD)
-   他们试图实现的内核进展是什么？
-   他们“雇佣”你的产品去完成什么“任务”？

### 4. 需求与痛点
-   他们最迫切的问题是什么？
-   不解决这些问题的后果是什么？

## 工作流程

1.  **从客户访谈、调查和产品分析中收集数据**。
2.  **分析你的“最佳”客户**（LTV 最高、流失率最低、成交最快）。
3.  **寻找他们在人口统计、行为和需求方面的模式**。
4.  **将这些模式综合**成一份清晰的 ICP 文档。
5.  **向销售、营销和产品团队验证 ICP**。
6.  **使用 ICP** 来指导产品开发和 GTM 工作。
