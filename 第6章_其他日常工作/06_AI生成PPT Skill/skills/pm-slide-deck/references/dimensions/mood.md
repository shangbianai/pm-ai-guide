# 氛围维度

色彩温度与调色板风格。

## 可选值

| 选项 | 色彩温度 | 调色板风格 | 最适合场景 |
|--------|-------------------|---------------|----------|
| `professional` | 冷-中性 | 海军蓝、金色、结构化灰色 | 商业、投资人、企业 |
| `warm` | 暖色 | 大地色、橙色、自然色 | 教育、友好、亲切 |
| `macaron` | 暖色 | 奶油底上的柔和色块（蓝色、薄荷绿、薰衣草、蜜桃色） | 教育图表、知识分享 |
| `cool` | 冷色 | 蓝色、灰色、青色、蓝绿 | 技术、数据、分析 |
| `vibrant` | 多变 | 高饱和度、大胆色彩 | 营销、创意、吸引注意力 |
| `dark` | 深色 | 深色背景配明亮强调色 | 娱乐、游戏、氛围感 |
| `neutral` | 中性 | 极简色彩、灰度为主 | 高管、极简、精致 |

## 渲染指南

### professional

- 克制使用强调色
- 金色仅用于重点突出
- 干净的机构感
- 均衡的对比度

### warm

- 大量使用暖色调
- 自然、亲切的色彩
- 色彩间柔和过渡
- 温馨友好的氛围

### macaron

- 柔和的粉色色块作为独立信息区域
- 每个区域使用一种马卡龙色（蓝色、薄荷绿、薰衣草、蜜桃）
- 珊瑚红强调色谨慎使用以突出重点
- 暖奶油色背景保持友好、亲切的感觉

### cool

- 蓝色主导的调色板
- 色彩使用体现技术精确感
- 高对比度确保清晰度
- 分析感、可信赖感

### vibrant

- 大胆的色彩组合
- 全局高饱和度
- 动感的色彩对比
- 充满活力的视觉表现

### dark

- 深色背景主导
- 强调色在深色背景上跳跃
- 发光/荧光效果
- 电影般的氛围

### neutral

- 极少的色彩使用
- 字体承担视觉重量
- 灰度层级
- 最大化的精致感

## 组合建议

| 氛围 | 最佳搭配 | 应避免搭配 |
|------|-----------------|------------|
| professional | clean 纹理、geometric 字体 | organic 纹理、handwritten |
| warm | organic 纹理、humanist 字体 | pixel 纹理、minimal 密度 |
| macaron | organic 纹理、handwritten 字体 | pixel 纹理、dark 氛围 |
| cool | grid 纹理、technical 字体 | paper 纹理、handwritten |
| vibrant | pixel/organic 纹理、editorial 字体 | 与 neutral 氛围重叠 |
| dark | clean/pixel 纹理、technical 字体 | paper 纹理 |
| neutral | clean 纹理、geometric 字体 | organic 纹理、vibrant 元素 |
