# /analyze-test -- A/B 测试分析
评估您的实验结果，查看是否具有统计显著性，以及您应该发布、停止还是延长测试。

## 工作流程
### 第 1 步：输入测试数据
用户提供：
- **对照组和实验组数据**：样本量和转化次数（或连续指标的平均值和标准差）。
- **主要指标**：我们试图提升的目标。
- **护栏指标**：我们不想损害的指标。

### 第 2 步：执行统计分析
计算：
- **P 值**：结果是否具有统计显著性？
- **置信区间**：效应的可能范围是多少？
- **样本量验证**：测试运行时间是否足够长以达到所需的功效？

### 第 3 步：解读结果
- **结果**：发布 (Ship)、停止 (Stop) 或调查 (Investigate)。
- **关键要点**：数据反映出的用户行为。
- **后续步骤**：测试后的行动和建议。
