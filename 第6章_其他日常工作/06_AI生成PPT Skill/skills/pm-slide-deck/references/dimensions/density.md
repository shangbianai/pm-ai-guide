# 密度维度

每张幻灯片的信息密度。

## 可选值

| 选项 | 每页内容量 | 留白 | 最适合场景 |
|--------|---------------|------------|----------|
| `minimal` | 一个焦点 | 最大化 | 高管简报、主题演讲、情感冲击 |
| `balanced` | 2-3 个要点 | 标准 | 通用演示、混合受众 |
| `dense` | 多个数据点 | 紧凑 | 数据密集、技术评审、详细分析 |

## 渲染指南

### minimal

- 每页一个核心观点
- 大面积视觉元素主导
- 最少文字（标题 + 最多 1-2 行）
- 充足的边距（距边缘 15% 以上）
- 元素间最大化呼吸空间
- 让单一元素承载全部视觉重量

**原则**：
- "一页一信息"
- 视觉 > 文字
- 空白是有意为之
- 每个元素都必须有存在的理由

### balanced

- 每页 2-3 个要点
- 标准边距（距边缘 10%）
- 文字/视觉均衡比例
- 清晰的层级与支撑细节
- 舒适的阅读体验

**原则**：
- 主要观点 + 辅助上下文
- 视觉辅助文字
- 结构化但不拥挤
- 适合多样化受众

### dense

- 可容纳多个数据点
- 紧凑边距（距边缘 5-8%）
- 信息丰富的布局
- 图表、表格、详细标注
- 假设受众专注且投入

**原则**：
- 数据胜于装饰
- 标注解释洞察
- 留白是策略性的
- 每个像素都有用途

## 受众 → 密度映射

| 受众 | 推荐密度 |
|----------|-------------------|
| 高管 | minimal |
| 初学者 | minimal 至 balanced |
| 通用 | balanced |
| 中级 | balanced |
| 专家 | balanced 至 dense |

## 幻灯片类型 → 密度指南

| 幻灯片类型 | 推荐密度 |
|------------|-------------------|
| 封面/标题 | minimal |
| 章节分隔 | minimal |
| 引言/冲击 | minimal |
| 议程/概览 | balanced |
| 内容/分析 | balanced 或 dense |
| 总结/要点 | balanced |
| 数据/指标 | dense |

## 各密度的内容指南

### minimal

| 元素 | 指南 |
|---------|-----------|
| 标题 | 大号（相当于 40-60pt） |
| 正文 | 极少或不使用 |
| 项目符号 | 最多 0-2 个 |
| 视觉元素 | 1 个主导元素 |
| 图表/数据 | 仅 1 个关键数据 |

### balanced

| 元素 | 指南 |
|---------|-----------|
| 标题 | 中大号（相当于 32-48pt） |
| 正文 | 2-4 行 |
| 项目符号 | 2-4 个 |
| 视觉元素 | 1-2 个 |
| 图表/数据 | 可使用简单图表 |

### dense

| 元素 | 指南 |
|---------|-----------|
| 标题 | 中号（相当于 24-36pt） |
| 正文 | 可使用多段落 |
| 项目符号 | 4-6 个以上 |
| 视觉元素 | 允许多个 |
| 图表/数据 | 可使用复杂图表、表格 |

## 组合建议

| 密度 | 最佳搭配 | 应避免搭配 |
|---------|-----------------|------------|
| minimal | neutral 氛围、geometric 字体 | 数据密集型内容 |
| balanced | 任何氛围/字体 | 极端情况（过于稀疏或过于拥挤） |
| dense | cool 氛围、technical 字体 | handwritten 字体、organic 纹理 |
