# 竞争对比卡 (Competitive Battlecard)

为销售和产品团队提供一份关于如何战胜特定竞争对手的简洁指南。重点关注关键差异点、“如何获胜”以及“设置陷阱”。

## 核心组成部分

### 1. 竞争对手概况
-   **他们是谁：** 他们的规模、市场地位和核心关注点。
-   **他们的“初衷”：** 他们的使命是什么？他们重视什么？

### 2. 关键差异点 (为什么我们能赢)
-   列出你的产品相对于他们产品的 3-5 个明确优势。
-   关注结果和用户利益，而不仅仅是功能。

### 3. 常见异议 (为什么他们能赢)
-   竞争对手通常在哪些方面击败我们？
-   我们应该如何回应这些异议？

### 4. “设置陷阱” (Traps to Set)
-   我们应该鼓励潜在客户向竞争对手提出哪些能暴露其弱点的问题？

### 5. 定价与包装
-   他们的定价如何对比？
-   我们在哪里可以提供更好的价值？

## 工作流程

1.  **选择一个关键竞争对手**进行分析。
2.  **研究他们的产品、定价和评论**。
3.  **访谈销售/客户成功团队**，了解常见的异议以及赢单/输单案例。
4.  **将信息综合**成对比卡格式。
5.  **与关键利益相关者一起审查并完善**。
6.  **分发并培训**团队如何使用它。
