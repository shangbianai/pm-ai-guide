# 纹理维度

视觉纹理与背景处理。

## 可选值

| 选项 | 背景 | 视觉元素 | 最适合场景 |
|--------|------------|-----------------|----------|
| `clean` | 纯色，无纹理 | 干净线条、几何形状 | 高管、极简、企业 |
| `grid` | 淡淡的网格叠加 | 网格线、示意图、技术图表 | 技术、架构、工程 |
| `organic` | 柔和纹理、手绘感 | 笔触、水彩、素描线条 | 创意、教育、友好 |
| `pixel` | 粗像素、8-bit 风格 | 像素画、复古游戏元素 | 游戏、开发者、怀旧 |
| `paper` | 做旧/质感纸张 | 复古元素、印章、斑驳 | 历史、传承、叙事 |

## 渲染指南

### clean

- 纯色背景，无可见纹理
- 所有元素边缘锐利清晰
- 数码精度与清晰度
- 最大化可读性对比度

### grid

- 淡淡的网格叠加（5-10% 不透明度）
- 工程绘图纸或蓝图感
- 对齐参考线可见但不突兀
- 技术制图美学

### organic

- 纸张纹理或画布质感
- 不完美边缘、自然变化
- 手绘色彩填充
- 随性、亲切的感觉

### pixel

- 可见的像素网格（粗犷而非精细）
- 8-bit 色彩风格
- 锯齿边缘（无平滑处理）
- 复古游戏 UI 元素

### paper

- 做旧纸张纹理（细微折痕、泛黄）
- 复古印刷痕迹
- 褐色或暖色调
- 历史文档感

## 组合建议

| 纹理 | 最佳搭配 | 应避免搭配 |
|---------|-----------------|------------|
| clean | professional、neutral 氛围 | handwritten 字体 |
| grid | cool、professional 氛围 | handwritten、vibrant 氛围 |
| organic | warm、macaron、vibrant 氛围 | technical 字体 |
| pixel | vibrant、dark 氛围 | editorial 字体 |
| paper | warm 氛围 | geometric 字体、minimal 密度 |
