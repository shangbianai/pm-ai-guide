安装

安装 arch-diagram Skill

1

复制 SKILL.md 到 skills 目录

bash
# Mac / Linux cp skill/SKILL.md ~/.claude/skills/arch-diagram/SKILL.md # Windows (PowerShell) Copy-Item skill\SKILL.md "$env:USERPROFILE\.claude\skills\arch-diagram\SKILL.md"
2

重启 Claude Code

关闭并重新打开 Claude Code，Skill 会自动加载。你会在技能列表中看到 arch-diagram。

3

触发 Skill

claude code
/arch-diagram

或者直接描述需求，Claude 会自动识别并激活：

claude code
帮我画一个电商系统的架构图
使用流程

三步完成架构图

1

提供架构信息（任意一种方式）

Skill 会优先从当前上下文提炼，不需要你重新描述：

方式一 · 直接描述
系统名：订单管理系统 模块：用户 App、API 网关、订单服务、库存服务、MySQL、Redis、消息队列 连接：App → 网关 → 订单服务 → MySQL / Redis 订单服务 → 消息队列 → 库存服务
方式二 · 粘贴已有文档
（直接粘贴 PRD / 技术方案 / 代码文件） Skill 自动提炼架构信息，告知你识别到了什么
2

选择输出模式

Skill 提示
请选择输出模式： A) SVG 直接生成 — 精准 HTML 文件，浏览器打开即用 B) AI 图片模式 — 生成提示词 / 调用 ChatGPT Image API
3

获取输出，持续迭代

路径 A 会在当前目录生成 xxx-architecture.html，用浏览器打开。

路径 B 会输出中英双语提示词，直接复制使用。

提示：可以继续对话迭代，例如"把数据库改成 PostgreSQL"、"加一个 CDN 节点"、"调整一下布局"，Claude 会直接修改文件。

路径 B · AI 图片模式

三种视觉风格

A

深色科技风（推荐）

与 ChatGPT Image 2 配合效果最佳，生成结果接近 AWS 架构图风格。

B

白底简洁风

适合插入 PPT、Word 文档，Google Cloud 架构图风格，背景干净。

C

手绘白板风

Excalidraw 草图风格，适合头脑风暴展示和非正式场合。

配置 API 直接生成图片

未配置 → 输出提示词，手动粘贴到 ChatGPT
已配置 OPENAI_API_KEY → 一键调用 gpt-image-1，直接输出图片文件
Claude Code 中运行
# 方法一：通过 Skill 配置（推荐） /update-config # 然后告诉它：set OPENAI_API_KEY=sk-... # 方法二：当次会话临时设置 ! $env:OPENAI_API_KEY="sk-..."

注意：gpt-image-1 需要 OpenAI 账号完成付款验证（Tier 1+），免费账号无法调用。

工具对比

产品架构图工具全景

工具 精度 速度 学习成本 适用场景
/arch-diagram（路径A） ★★★★★ ★★★★★ 极低（描述即可） 正式交付、文档归档、PPT 插图
ChatGPT Image 2（路径B） ★★★ ★★★★ 低 课程演示、探索视觉风格
Mermaid / D2 ★★★★ ★★★ 中（需学语法） 代码仓库 README、技术文档
draw.io / Lucidchart ★★★★★ ★★ 高 需精细手工调整的场景
Excalidraw ★★★ ★★★ 低 头脑风暴、草图沟通
Figma / FigJam ★★★★★ ★★ 高 团队协作、设计规范沉淀

PM 选型建议：日常工作优先用路径 A，速度快且结果专业；需要展示 AI 图片生成能力或探索风格时用路径 B；需要团队在线协作编辑时才考虑 Figma/draw.io。
