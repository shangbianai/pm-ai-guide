# 04-PM市场研究 (PM Market Research)

产品经理市场研究技能库：涵盖用户画像、市场细分、情感分析和竞争对手分析。

## 技能 (Skills) (7)

- **competitor-analysis** — 分析竞争对手的优势、劣势和差异化机会。
- **customer-journey-map** — 创建端到端的客户旅程地图，包含阶段、接触点、情感、痛点和机会。
- **market-segments** — 识别 3-5 个潜在客户细分市场，并进行人口统计、JTBD 和产品拟合度分析。
- **market-sizing** — 使用 TAM、SAM 和 SOM 以及自上而下和自下而上的方法估算市场规模。
- **sentiment-analysis** — 分析用户反馈数据，通过情感评分、JTBD 和产品满意度洞察来识别市场细分。
- **user-personas** — 根据研究数据创建精炼的用户画像。
- **user-segmentation** — 根据行为、JTBD 和需求从反馈数据中对用户进行细分。

## 工作流 (Workflows) (3)

- `分析用户反馈.md` — 大规模分析用户反馈 —— 情感分析、主题提取和细分层面的洞察。
- `用户研究.md` — 全面的用户研究 —— 构建画像、细分用户并从研究数据中绘制客户旅程图。
- `竞争分析.md` — 分析竞争格局 —— 识别竞争对手，比较优势和劣势，寻找差异化机会。
